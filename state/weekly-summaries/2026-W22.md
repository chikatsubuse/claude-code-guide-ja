# 週次サマリ 2026-W22

## 🔴 Breaking Changes

- **`/simplify` → `/code-review`** (v2.1.147): コマンドが改称され、旧 cleanup-and-fix 動作は廃止。新コマンドは effort 指定 (`/code-review high`) でバグを検出し、`--comment` フラグで GitHub PR のインラインコメントとして投稿できる
- **`/model` が現セッション限定に変更** (v2.1.144): `/model` はデフォルトを変えなくなった。デフォルトモデルの変更は model picker 内で `d` キーを押す

## 🟠 Major Updates

- **`claude agents` Agent View (Research Preview)** (v2.1.139): 実行中・待機中・完了済みのすべての Claude Code セッションを 1 画面で管理できる UI。`claude agents --json` でスクリプト向け JSON 出力も可 (v2.1.145)
- **`/goal` コマンド** (v2.1.139): 完了条件を設定すると Claude がその条件を満たすまでターンを継続。インタラクティブ・`-p` モード・Remote Control で動作
- **`claude plugin details <name>`** (v2.1.139): プラグインのコンポーネント一覧と推定トークンコストを CLI から確認
- **Hook exec form (`args: string[]`)** (v2.1.139): シェルを介さずコマンドを直接起動。パスにスペースが含まれても安全
- **Hook `continueOnBlock`** (v2.1.139): `PostToolUse` フックのブロック後もターンを継続し、ブロック理由を Claude に伝える
- **`terminalSequence` hook フィールド** (v2.1.141): ヘッドレス環境でもデスクトップ通知・ウィンドウタイトル変更可
- **プラグイン依存管理強制** (v2.1.143): `claude plugin disable` は依存元があれば拒否。`enable` は依存プラグインを自動有効化
- **`/resume` background sessions 対応** (v2.1.144): `claude --bg` で起動したセッションも `/resume` 対象に。選択肢に `bg` マーカー付き

## 🟡 Minor Updates

- `/usage` per-category 内訳 (v2.1.149): skills / subagents / plugins / MCP サーバー別のコスト内訳を表示
- `worktree.bgIsolation: "none"` (v2.1.143): バックグラウンドセッションが EnterWorktree なしで作業コピーを直接編集できる
- Rewind "Summarize up to here" (v2.1.141): 選択時点までを圧縮しつつ新しいターンを保持
- Pinned background sessions (v2.1.147): `Ctrl+T` でセッションをピン留めするとアイドル中も生存
- `/plugin` Discover/Browse がインストール前にコンポーネント一覧・推定コストを表示 (v2.1.143/v2.1.145)
- root-level `SKILL.md` が Skill として認識 (v2.1.142): `skills/` サブディレクトリ不要
