# 週次サマリ 2026-W20

## 🔴 Breaking Changes

なし

## 🟠 Major Updates

- **`autoMode.hard_deny`** (v2.1.136): Auto Mode に例外なしの無条件ブロックルールを追加する設定が追加された。`soft_deny` との違いは、ユーザーが明示的に許可しても通らない点。→ `guide/05-auto-mode-computer-use.md`
- **`worktree.baseRef` 設定** (v2.1.133): worktree の分岐元を `fresh` (origin/<default>) か `head` (ローカル HEAD) で選べるようになった。デフォルトは `fresh` で、v2.1.128 で local HEAD に変わっていた挙動を元に戻す。→ `guide/10-parallel-execution.md`
- **Hooks が effort level を受け取れる** (v2.1.133): Hooks の JSON 入力に `effort.level` フィールドが追加。Bash ツール内でも `$CLAUDE_EFFORT` で参照可能。→ `guide/06-extension-mechanisms.md`
- **MCP Tool hook** (v2.1.118): Hook から MCP サーバーのツールを直接呼び出せる `type: "mcp_tool"` が追加された。→ `guide/06-extension-mechanisms.md`
- **PostToolUse が全ツール対応** (v2.1.121): `hookSpecificOutput.updatedToolOutput` で全ツールの出力を書き換えられるようになった (従来は MCP のみ)。→ `guide/06-extension-mechanisms.md`
- **`alwaysLoad` MCP オプション** (v2.1.121): MCP サーバーのツールを tool-search 遅延なく常時ロードする設定。→ `guide/06-extension-mechanisms.md`
- **`--plugin-url <url>` フラグ** (v2.1.129): URL から plugin .zip を直接フェッチしてセッション利用。→ `guide/07-plugin-marketplace.md`
- **`--plugin-dir` が .zip 対応** (v2.1.128): ディレクトリだけでなく .zip アーカイブを直接指定可能に。→ `guide/07-plugin-marketplace.md`
- **`claude plugin prune`** (v2.1.121): 孤立した依存プラグインを削除するサブコマンド追加。→ `guide/07-plugin-marketplace.md`
- **vim visual mode** (v2.1.118): `v` でビジュアル選択、`V` でビジュアル行選択モードが追加。→ `guide/11-keybindings-cheatsheet.md`
- **`/usage` 統合** (v2.1.118): `/cost` と `/stats` が `/usage` に統合 (旧コマンドはショートカットとして継続)。→ `guide/11-keybindings-cheatsheet.md`
- **カスタムテーマ** (v2.1.118): `/theme` でカスタムテーマの作成・切替が可能に。→ `guide/02-setup.md`
- **デフォルト effort が `high` に** (v2.1.117): Pro/Max ユーザーの Opus 4.6・Sonnet 4.6 のデフォルト effort が `medium` から `high` に引き上げ。→ `guide/02-setup.md`
- **`claude ultrareview [target]`** (v2.1.120): `/ultrareview` を非対話で CI・スクリプトから実行できるサブコマンド追加。→ `guide/11-keybindings-cheatsheet.md`, `guide/appendix-b-commands.md`
- **`--channels` コンソール認証対応** (v2.1.128): `--channels` フラグが API キー認証でも動作するように。→ `guide/10-parallel-execution.md`

## 🟡 Minor Updates

- **Windows: Git for Windows 不要** (v2.1.120): Git Bash がない場合は PowerShell をシェルとして自動使用。→ `guide/02-setup.md`
- **`${CLAUDE_EFFORT}` プレースホルダー** (v2.1.120): Skill 本文内で現在の effort level を参照可能に。→ `guide/06-extension-mechanisms.md`
- **`claude plugin tag`** (v2.1.118): プラグインのリリース Git タグをバージョン検証付きで作成。→ `guide/07-plugin-marketplace.md`
- **`sandbox.bwrapPath`/`socatPath`** (v2.1.133): bubblewrap/socat のカスタムバイナリパス設定 (Linux/WSL)。→ `guide/05-auto-mode-computer-use.md`
- **`claude project purge`** (v2.1.126): プロジェクトの Claude Code 状態を全削除するコマンド。`--dry-run` 対応。→ `guide/appendix-b-commands.md`
