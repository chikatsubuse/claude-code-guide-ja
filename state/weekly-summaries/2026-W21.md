# 週次サマリ 2026-W21

## 🔴 Breaking Changes
なし

## 🟠 Major Updates
- **Agent ビュー** (v2.1.139): `claude agents` で実行中・待機中・完了済みの全セッションを一画面に表示するダッシュボード (リサーチプレビュー)。`--cwd` による絞り込み、`--model`/`--effort`/`--permission-mode` 等のフラグで派生セッションのデフォルトを設定可能 (v2.1.142〜v2.1.143)
- **`/goal` コマンド** (v2.1.139): 完了条件を設定すると Claude が条件達成まで自律的にターンを継続。対話・`-p`・Remote Control で動作。経過時間/ターン数/トークンをオーバーレイ表示
- **Plugin 依存強制** (v2.1.143): `claude plugin disable` が他プラグインへの依存を検出して拒否。`claude plugin enable` が推移的依存関係を自動有効化
- **`/plugin` Projected Cost** (v2.1.143): マーケットプレイスの Browse ペインにプラグインのターンごと・起動ごとのトークン推定コストを表示
- **`worktree.bgIsolation: "none"`** (v2.1.143): worktree が利用できないリポジトリでバックグラウンドセッションが作業ツリーを直接編集可能に
- **Fast モード Opus 4.7 デフォルト** (v2.1.142): Fast モードのデフォルトモデルが Opus 4.6 から Opus 4.7 に変更
- **Hook `terminalSequence`** (v2.1.141): フック出力からデスクトップ通知・ウィンドウタイトル・ベル音を発行可能に

## 🟡 Minor Updates
- **Hook `args: string[]`** (v2.1.139): exec 形式でシェル不要のコマンド直接起動
- **Hook `continueOnBlock`** (v2.1.139): `PostToolUse` で拒否時にターンを継続させる新オプション
- **MCP `CLAUDE_PROJECT_DIR`** (v2.1.139): MCP stdio サーバーの環境に `CLAUDE_PROJECT_DIR` が自動挿入
- **Plugin root-level SKILL.md** (v2.1.142): `skills/` サブディレクトリなしでルートの `SKILL.md` が Skill として認識
- **`claude plugin details`** (v2.1.139): プラグインのコンポーネント一覧と推定トークンコストを表示
- **`/scroll-speed`** (v2.1.139): マウスホイールスクロール速度をライブプレビュー付きで調整
- **Background session: model/effort 保持** (v2.1.143): アイドルから復帰後もモデルと effort が維持
- **`/bg` フラグ保持強化** (v2.1.143): `--mcp-config`/`--settings`/`--add-dir`/`--plugin-dir`/`--fallback-model`/`--allow-dangerously-skip-permissions` を保持
