# 週次サマリ 2026-W16

## 🔴 Breaking Changes

なし。

## 🟠 Major Updates

- **Claude Opus 4.7** (v2.1.111, 2026-04-16): 最新フラッグシップモデル。`xhigh` effort level を追加し、`high` と `max` の間の中間深度を提供。Max サブスクライバー向けに Auto モードが GA。
- **CLI ネイティブバイナリ化** (v2.1.113): CLI がプラットフォームごとのオプション依存としてパッケージされたネイティブバイナリを起動するようになった。従来の JavaScript バンドル実行方式より起動速度・メモリ効率が向上する。

## 🟡 Minor Updates

- **`/effort` が対話スライダー化** (v2.1.111): 引数なしで実行すると矢印キーで選択可能。
- **Auto (match terminal) テーマ** (v2.1.111): ターミナルの dark/light モードに追従する `/theme` オプション。
- **`/less-permission-prompts` Skill** (v2.1.111): 過去トランスクリプトから読み取り専用 Bash/MCP 呼び出しを分析し、`.claude/settings.json` の allowlist 候補を提案。
- **`sandbox.network.deniedDomains` 設定追加** (v2.1.113): `allowedDomains` ワイルドカードで許可されたドメインのうち特定ドメインだけをブロックできる。
- **`Ctrl+A`/`Ctrl+E` キーバインド追加** (v2.1.113): 複数行入力で論理行の行頭/行末に移動 (readline 互換)。
- **`/extra-usage` が Remote Control から利用可能** (v2.1.113): スマホ/Web クライアントからも `@`-ファイル補完やコマンド実行が可能に。
