# 週次サマリ 2026-W19

## 🔴 Breaking Changes
なし

## 🟠 Major Updates
- **claude project purge** (v2.1.126): プロジェクトのトランスクリプト・タスク・ファイル履歴・設定を一括削除。`--dry-run`/`-y`/`--interactive`/`--all` オプション付き
- **claude ultrareview** (v2.1.120): CI やスクリプトから `/ultrareview` を非対話実行するサブコマンド。`--json` で raw 出力、終了コード 0/1 で成否判定
- **MCP alwaysLoad** (v2.1.121): MCP サーバー設定に `alwaysLoad: true` を指定するとツール検索の遅延を経ずに常時利用可能になる
- **claude plugin prune** (v2.1.121): 孤立した自動インストール済み依存を一括削除。`plugin uninstall --prune` で連鎖削除も可能
- **PostToolUse で全ツール出力を置換** (v2.1.121): `hookSpecificOutput.updatedToolOutput` で MCP 以外のツールも出力を差し替え可能に
- **Vim visual mode** (v2.1.118): `v`/`V` でビジュアル選択・ビジュアルライン選択が可能に。オペレータ (`d`, `y`, `c` 等) を適用できる
- **/cost・/stats → /usage 統合** (v2.1.118): `/cost` と `/stats` を `/usage` タブに統合。旧コマンドは引き続き動作するショートカットとして維持
- **カスタムテーマ作成** (v2.1.118): `/theme` から名前付きカスタムテーマを作成・切替可能に。`~/.claude/themes/` へ JSON 手書きでも設定可。プラグインが `themes/` ディレクトリで配布も可能
- **Hooks から MCP ツール呼出** (v2.1.118): `type: "mcp_tool"` でフックから MCP ツールを直接呼び出せるようになった

## 🟡 Minor Updates
- **デフォルト effort が high に** (v2.1.117): Pro/Max プランの Opus 4.6・Sonnet 4.6 のデフォルト effort が `medium` → `high` に変更
- **${CLAUDE_EFFORT} プレースホルダ** (v2.1.120): Skill 本文内で現在の effort レベルを参照可能に
- **/skills 検索ボックス** (v2.1.121): `/skills` ダイアログにキーワード絞り込み検索を追加
- **PostToolUse に duration_ms** (v2.1.119): `PostToolUse`/`PostToolUseFailure` フック入力にツール実行時間 (ms) が追加
- **ANTHROPIC_BEDROCK_SERVICE_TIER** (v2.1.122): Bedrock のサービスティア (`default`/`flex`/`priority`) を環境変数で選択可能に
- **/model でゲートウェイモデル一覧取得** (v2.1.126): `ANTHROPIC_BASE_URL` がゲートウェイを指す場合、`/v1/models` からモデル一覧を自動取得
- **OAuth コードをターミナルに貼り付け** (v2.1.126): `claude auth login` で WSL2・SSH・コンテナ環境でもブラウザ不要のコード貼り付け認証をサポート
- **Windows: Git 不要** (v2.1.120): Git for Windows がなくても PowerShell をシェルとして使用可能に
