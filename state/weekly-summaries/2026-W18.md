# 週次サマリ 2026-W18

## 🔴 Breaking Changes
なし

## 🟠 Major Updates
- **Vim ビジュアルモード** (v2.1.118): `v` でビジュアル選択、`V` で行選択が可能に。選択・オペレータ・ビジュアルフィードバック対応
- **`/usage` 統合** (v2.1.118): `/cost` と `/stats` が `/usage` タブに統合。従来コマンドはショートカットとして継続動作
- **カスタムテーマ対応** (v2.1.118): `/theme` から名前付きカスタムテーマを作成・切替可能。`~/.claude/themes/` の JSON 編集にも対応。プラグインも `themes/` ディレクトリでテーマを同梱可能
- **Hooks から MCP ツール直接呼び出し** (v2.1.118): `type: "mcp_tool"` ハンドラ追加。Hooks ハンドラが4種類に
- **`claude plugin tag` 追加** (v2.1.118): バージョン検証付きでプラグインのリリース用 git タグを作成
- **デフォルト effort が `high` に変更** (v2.1.117): Pro/Max プランの Opus 4.6・Sonnet 4.6 のデフォルト effort が `medium` → `high` に

## 🟡 Minor Updates
- **`PostToolUse`/`PostToolUseFailure` に `duration_ms` 追加** (v2.1.119): ツール実行時間をフック入力で取得可能に
- **`--agent` フロントマター完全対応** (v2.1.116/117): `hooks:` (v2.1.116)・`mcpServers:` (v2.1.117) がメインスレッドエージェントでも動作
- **Auto Mode に `"$defaults"` サポート** (v2.1.118): カスタムルールにデフォルトリストを維持したまま追加できる
- **WSL `wslInheritsWindowsSettings`** (v2.1.118): Windows 側 managed settings を WSL 内で引き継ぐポリシーキー追加
- **`blockedMarketplaces` 適用範囲拡大** (v2.1.117): install/update/refresh/autoupdate の全操作に適用
- **Plugin 依存関係の自動補完** (v2.1.117): `plugin install` で既インストール済みプラグイン指定時も不足依存を自動取得
- **`/reload-plugins` の自動依存解決** (v2.1.116): 設定済みマーケットプレイスから不足依存を自動取得
- **`/config` 設定の永続化** (v2.1.119): theme・editor mode・verbose 等が `~/.claude/settings.json` に保存されプロジェクト/ローカル/ポリシーの優先度に参加
